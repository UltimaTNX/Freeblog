Welcome to Freeblog GitHub Pages.

Freeblog è un piccolo CMS ideato e sviluppato da Stefano Pascazi sulla base della propria esperienza lavorativa personale, per facilitare il lavoro di creazione applicativi ad-hoc. Non si appoggia a nessun framework di sviluppo. Tutte le Classi php contenute al suo interno, sono state ideate e sviluppate da Stefano Pascazi e sono state fatte in modo che risolvano qualsiasi tipo di problematica, scrivendo quanto meno codice possibile.

Support al Designer Templates

L'intero CMS comprende già un template default contenuto all'interno della cartella /template/ della root principale. Sulla base di quel template è possibile realizzarne altri. Basterà creare un template, utilizzando le marcature idonee per il richiamo dei file e dei dati, inserirlo in una cartella tipo: 'nome_template', aggiornare il file config.php indicando quale tipo di template utilizzare.

Come installare Freeblog?

Freeblog si installa in pochi e semplice passaggi. Richiede necessariamente un database con support Mysqli. Una volta creato il database:

  • Aprire il file config.php contenuto nella cartella /Classes/
  • Configurare secondo i parametri del proprio database
  • Salvare il file
  • Caricare tutto il pacchetto all'interno del proprio spazio hosting
  • Lanciare: http://www.tuo_sito.it/install/
  • Al temine dell'installazione, rimuovere la cartella /install/
